# Configuring Wind Network

Wind Network can be configured through a TOML configuration file. This guide explains all available configuration options and recommended settings.

## Configuration File

The default location for the configuration file is `/etc/wind/config.toml` (Linux/macOS) or `C:\ProgramData\Wind\config.toml` (Windows).

You can specify a custom configuration file location with the `--config` flag:

```bash
wind-node --config /path/to/config.toml
```

## Basic Configuration

Here's a minimal configuration example:

```toml
[node]
name = "my-wind-node"
log_level = "info"

[network]
listen_addr = "0.0.0.0:9000"
external_addr = "203.0.113.1:9000"  # Your public IP if behind NAT
bootstrap_nodes = [
  "/dns4/bootstrap1.wind.network/tcp/9000/p2p/QmcZf59bWwK5XFi76CZX8cbJ4BhTzzA3gU1ZjYZcYW3dwt",
  "/dns4/bootstrap2.wind.network/tcp/9000/p2p/QmbBHw1Xx9pUpAbrVZUKTPL5Rsph5Q9GQhRvcWVBPFgGtC"
]

[geyser]
plugin_path = "/usr/local/lib/libwindexer_geyser.so"
rpc_url = "https://api.mainnet-beta.solana.com"

[storage]
data_dir = "/var/lib/wind"
db_engine = "rocksdb"
```

## Advanced Configuration

### Network Settings

```toml
[network]
listen_addr = "0.0.0.0:9000"
external_addr = "203.0.113.1:9000"
bootstrap_nodes = [
  "/dns4/bootstrap1.wind.network/tcp/9000/p2p/QmcZf59bWwK5XFi76CZX8cbJ4BhTzzA3gU1ZjYZcYW3dwt",
  "/dns4/bootstrap2.wind.network/tcp/9000/p2p/QmbBHw1Xx9pUpAbrVZUKTPL5Rsph5Q9GQhRvcWVBPFgGtC"
]
max_connections = 50
connection_timeout_sec = 30
dht_randomwalk = true
mdns_enabled = true
```

### Geyser Plugin Settings

```toml
[geyser]
plugin_path = "/usr/local/lib/libwindexer_geyser.so"
rpc_url = "https://api.mainnet-beta.solana.com"
websocket_url = "wss://api.mainnet-beta.solana.com"
account_subscriptions = ["TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA"]
transaction_subscriptions = ["vote"]
```

### Storage Settings

```toml
[storage]
data_dir = "/var/lib/wind"
db_engine = "rocksdb"  # Options: "rocksdb", "sqlite", "postgres"
max_open_files = 1000
write_buffer_size = "64MB"
max_write_buffer_number = 4
keep_log_file_num = 10

# PostgreSQL-specific settings (if db_engine = "postgres")
postgres_url = "postgres://username:password@localhost/windnetwork"
```

### API Settings

```toml
[api]
enabled = true
listen_addr = "127.0.0.1:8080"
cors_allowed_origins = ["*"]
rate_limit = 100  # requests per minute
jwt_secret = ""  # Leave empty for no authentication
```

### Jito Staking

```toml
[jito]
enabled = true
stake_amount = "10000"  # in SOL
validator_identity = "/path/to/validator-keypair.json"
rewards_address = "AKTTxGqJPkJGC6DKvSmMUGXtKMuBnTWnGEEeU3PEwuTY"
```

## Environment Variables

You can also configure Wind Network using environment variables. Each configuration option can be set using the format `WIND_SECTION_KEY`:

```bash
export WIND_NODE_NAME="my-wind-node"
export WIND_NETWORK_LISTEN_ADDR="0.0.0.0:9000"
export WIND_STORAGE_DATA_DIR="/var/lib/wind"
```

Environment variables take precedence over configuration file settings.
