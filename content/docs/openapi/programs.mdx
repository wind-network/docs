# Program Endpoints

These endpoints allow you to fetch information about Solana programs indexed by Wind Network.

## Get Program Information

```
GET /v1/programs/{address}
```

Retrieves detailed information about a Solana program by its address.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| address | string | The program address (base58 encoded) |

### Response

**200 OK**

```json
{
  "address": "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA",
  "owner": "BPFLoaderUpgradeab1e11111111111111111111111",
  "executable": true,
  "data_size": 54321,
  "is_native": false,
  "lamports": 10290815,
  "rent_epoch": 327,
  "last_deployed_slot": 152678924,
  "last_deployed_time": "2023-01-15T12:30:45Z",
  "name": "Token Program",
  "description": "Solana Program Library (SPL) Token Program",
  "updated_at": "2023-05-15T14:32:10Z"
}
```

**404 Not Found**

```json
{
  "error": "Program not found",
  "code": 404
}
```

### Example

```bash
curl -X GET "https://api.wind.network/v1/programs/TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA" \
  -H "X-API-Key: your_api_key_here"
```

## List Program Accounts

```
GET /v1/programs/{address}/accounts
```

Lists accounts owned by a specific program.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| address | string | The program address (base58 encoded) |

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| limit | integer | Number of results per page (default: 50, max: 1000) |
| cursor | string | Cursor for fetching the next page |
| encoding | string | Data encoding to return (`base58`, `base64`, `jsonParsed`) (default: `jsonParsed`) |
| commitment | string | The commitment level: `processed`, `confirmed`, or `finalized` (default: `finalized`) |
| filters | array | JSON-encoded filter objects (see filter format below) |

#### Filter Format

```json
[
  {
    "memcmp": {
      "offset": 0,
      "bytes": "base58_or_base64_encoded_data"
    }
  },
  {
    "dataSize": 128
  }
]
```

### Response

**200 OK**

```json
{
  "accounts": [
    {
      "address": "8VRnd8ioFKpNpBznCEFd6uiT55P4Zy1XgF6Jj8Efc7vY",
      "lamports": 5000000,
      "owner": "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA",
      "executable": false,
      "rent_epoch": 327,
      "data": "...",
      "data_base64": "...",
      "parsed_data": {
        // Program-specific parsed data if available
      },
      "updated_at": "2023-05-15T14:32:10Z"
    }
    // More accounts...
  ],
  "pagination": {
    "limit": 50,
    "next_cursor": "eyJhZGRyZXNzIjoiOFZSbmQ4aW9GS3BOcEJ6bkNFRmQ2dWlUNTVQNFp5MVhnRjZKajhFZmM3dlkifQ=="
  }
}
```

### Example

```bash
curl -X GET "https://api.wind.network/v1/programs/TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA/accounts?limit=10" \
  -H "X-API-Key: your_api_key_here"
```

## Get Program Transaction Count

```
GET /v1/programs/{address}/transaction-count
```

Returns the total number of transactions involving a specific program.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| address | string | The program address (base58 encoded) |

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| start_time | string | Filter by start time (ISO 8601 format) |
| end_time | string | Filter by end time (ISO 8601 format) |

### Response

**200 OK**

```json
{
  "program": "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA",
  "transaction_count": 98765432
}
```

### Example

```bash
curl -X GET "https://api.wind.network/v1/programs/TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA/transaction-count?start_time=2023-01-01T00:00:00Z&end_time=2023-02-01T00:00:00Z" \
  -H "X-API-Key: your_api_key_here"
```
